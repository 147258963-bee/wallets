# WalletsPoints 🔥 
### Sistema de Fidelización con QR y KPI en Tiempo Real

**WalletsPoints** es una solución tecnológica diseñada para comercios que buscan digitalizar sus sistemas de puntos. Permite a los negocios generar códigos QR únicos, registrar escaneos de clientes y visualizar métricas clave (KPIs) desde un panel móvil moderno.

---

## 🚀 Características Principales
- **Dashboard Mobile-First:** Interfaz optimizada para teléfonos inteligentes (UI tipo App).
- **Generación de QR Dinámico:** QR único por comercio y sector.
- **Seguridad:** Autenticación mediante **JWT** (JSON Web Tokens) y encriptación de claves con **Bcrypt**.
- **Métricas en Tiempo Real:** Visualización de lecturas totales y lecturas del día actual.
- **Arquitectura Unificada:** Backend (Node.js) y Frontend (HTML/Tailwind) en un solo ecosistema.

---

## 🛠️ Tecnologías Utilizadas
* **Backend:** Node.js & Express
* **Base de Datos:** MongoDB
* **Frontend:** HTML5, Tailwind CSS, JavaScript (Vanilla)
* **Seguridad:** JWT & Bcryptjs
* **Utilidades:** QRCode para generación de imágenes
