# 📱 Wallet Points | Business Intelligence System

Sistema de gestión de fidelización de clientes diseñado para dispositivos móviles. Permite el registro de miembros, acumulación de puntos mediante escaneo QR, gestión de catálogo de productos y envío de campañas por WhatsApp.

## 🚀 Funcionalidades Principales

* **Acceso Seguro:** Panel protegido por contraseña para el administrador.
* **Registro de Clientes:** Captura de nombre completo, RUT y WhatsApp con aceptación de políticas de privacidad.
* **Dashboard en Tiempo Real:** Visualización de total de clientes inscritos y puntos totales repartidos.
* **Catálogo de Productos:** Gestión de hasta 1,000 productos con descripción y precio.
* **Marketing Masivo:** Envío de ofertas personalizadas a toda la base de datos vía WhatsApp.
* **Fidelización:** Suma de puntos rápida para premiar la recurrencia.

## 🔐 Seguridad y Acceso

El sistema cuenta con un bloqueo de acceso para evitar que los clientes vean la base de datos o el catálogo.

## 🛠️ Instalación y Uso en Celular

Para usar esta aplicación como una herramienta de gestión en tu local:

1.  Sube el archivo `index.html` a un repositorio de **GitHub**.
2.  Activa **GitHub Pages** en `Settings > Pages`.
3.  Abre el enlace generado en el navegador de tu celular.
4.  **Tip:** Selecciona "Añadir a la pantalla de inicio" en el menú de Chrome/Safari para usarla como una App nativa.

## 📊 Almacenamiento de Datos

> [!IMPORTANT]  
> Esta aplicación utiliza **LocalStorage**. Los datos se guardan directamente en el navegador del dispositivo donde se registran. Se recomienda usar el botón **"Exportar"** regularmente para descargar un respaldo en formato `.json`.

## 🛠️ Tecnologías utilizadas

* **Tailwind CSS:** Diseño moderno y responsive.
* **Html5-QRCode:** Motor de escaneo de cámara.
* **JavaScript (ES6+):** Lógica de negocio y persistencia local.
* **Google Fonts (Outfit):** Tipografía corporativa.
